This is a sample description file for the FAIR Research tutorial: BDBags and Minids. 
It is meant to be included as a payload file in a BDBag instance along with the file "fair-research.jpg".
Both this file and "fair-research.jpg" have Minids assigned and are remotely referenced in the bag using these identifiers.

For more information see: 
https://github.com/fair-research/tutorials/bdbag-minid
